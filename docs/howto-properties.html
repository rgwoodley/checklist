Checklist : How-To Add Configuration Properties

This section explains how to add configuration properties to the checklist.properties that can then be accessed by classes in the Application.

  • Add the property to checklist.properties.
  • Add the property to the ApplicationProperties class in the com.codeapes.checklist.util package, along with getter/setter methods.
  • Add the property to the applicationProperties bean definition in the src/main/resources/checklist-props.xml file.
  • Use the property through the accessor method in the ApplicationProperties class; use it as a Spring-managed bean (don't create a new instance of it).
