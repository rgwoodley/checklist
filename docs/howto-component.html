Checklist : How to add a miscellaneous Spring component

It might be useful to add classes that are managed by Spring that don't fall into the category of a DAO, Service or Controller.

Use the @Component annotation
Annotate the class with @Component, and make sure the its containing Java package is covered in the base-package property of the context defined in src/main/resources/spring-config/checklist-service.xml.
Example

@Component("myComponent")
public class MyComponent {

    public void doSomething() {
        ....
    }
}

---

public class UsesMyComponent {

    @Autowired
    private MyComponent myComponent;

}
Notes
  • No additional XML configuration is needed to add a Component.
  • To use another Spring-managed component, use the @Autowired annotation. The @Autowired annotation can be paired with the @Qualifier annotation in case two components implement the same interface and spring does not select the correct bean. See the PersistenceDAOImpl.java source code for an example of this.
